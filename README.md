# Simple-Personal-Assistant
This repository is for a personal assistant named Buzzy!

Buzzy can tell you the current time. She can also play songs from Youtube, search anything from Wikipedia and tell you a joke! 

Just tell her to do things using 'time', 'search' or 'play' to know the time, search on Wiki and play from youtube respectively. She can also tell you a joke.

Example commands
1. Buzzy what is the current time / Buzzy tell me the time.
2. Buzzy search for Bangladesh.
3. Buzzy play Taylor Swift.
4. Buzzy tell me a joke.
